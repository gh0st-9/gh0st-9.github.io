Evil MySQL Server

利用rogue_mysql_server构造恶意mysql服务器以任意读取客户端文件

配置好文件，启动服务器，客户端连接得到flag文件名，最后进行读取

Be-a-Language-Expert

考点为thinkphp lang rce，也是Thinkphp多语言功能导致的任意文件包含问题，具体信息参考Thinkphp 多语言 RCE - 跳跳糖 (tttang.com)

打开题目发现是thinkphp6

参考p牛的文章，利用PearCMD来实现最终rce，

发送请求包，讲shell.php写入/tmp目录下

然后进行文件包含

反弹shell获得flag

Spring4Shell：

CVE-2022-22965

题目存在.git泄露，利用工具恢复

此题需要通过泄露的信息确定shell的位置，可通过appBase来确定项目路径

获取到web项目路径chaitin

可直接利用Spring4ShellEXP,项目位置需要修改

Be-a-Wiki-Hacker

直接利用GitHub脚本rce即可https://github.com/crowsec-edtech/CVE-2022-26134/
